# プロダクト概要

DDD と CQRS/イベントソーシング、GraphQL を組み合わせ、AI 支援の仕様駆動開発でイベントソースドなバックエンドを構築するためのテンプレート/キット。

## コア能力

- ドメインイベント/集約のモデリングと実装の指針を提供する
- コマンド/クエリを分離した GraphQL API の土台を用意する
- イベントストアとリードモデル更新（RMU）を繋ぐ仕組みを備える
- 仕様 → 設計 → 実装の Kiro ワークフローを AI ツールで回せる
- 参照実装に基づく実運用寄りの構成を再利用できる

## 対象ユースケース

- DDD/CQRS/ES の実装を短期間で立ち上げたいチーム
- Event Storming で整理したドメインイベントを実装に落とし込みたい
- GraphQL でコマンド/クエリを統合した API を提供したい
- AI ツールと仕様駆動で機能開発を進めたい

## 価値提案

- 実証済みの CQRS/ES パターンをテンプレート化し、設計と実装の乖離を減らす
- コマンド/クエリ/リードモデル更新を分離し、拡張しやすい構造にする
- 仕様駆動と AI 活用を前提にした作業導線が揃っている
